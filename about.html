Carrier Gateway

A multi-carrier insurance quote aggregation API built with Go. Fan-out to multiple carriers concurrently, deduplicate results, and return the best quotes — sorted by price — in a single response.

Go 1.25 Hexagonal Architecture Prometheus Metrics PostgreSQL (optional)

⚡ What It Does

Carrier Gateway is an HTTP service that aggregates insurance quotes from multiple carrier backends. When a client sends a quote request specifying coverage lines (auto, homeowners, umbrella), the gateway:

POST /quotes → Validate → Filter eligible carriers → Fan-out (parallel) → Collect & deduplicate → Sort by price → JSON response

Optionally caches quotes in PostgreSQL and returns cached results for repeated requests with the same request_id.

🏗 Architecture

Strict hexagonal (ports & adapters) layering. Import direction is one-way and never violated: inner layers know nothing about outer layers.

handler HTTP routes, validation, JSON
cmd/main.go Composition root
↓
orchestrator Fan-out, hedging, dedup
circuitbreaker 3-state CAS machine
ratelimiter Token bucket per carrier
adapter Type-erased registry
metrics Prometheus recorder
↓

↓
domain QuoteRequest, QuoteResult, Money, errors

✨ Key Features

⚡ Concurrent Fan-Out

Queries all eligible carriers in parallel using errgroup.WithContext. Results channel sized at 2×len(eligible) to handle primary + hedge per carrier without blocking.

🎯 Adaptive Hedging

EMA tracker per carrier computes p95 latency. When a carrier exceeds p95 × HedgeMultiplier, a hedge request fires automatically. Warm-up suppresses hedging for the first 10 observations.

🛡 Circuit Breaker

3-state machine (Closed / Open / HalfOpen) using atomic.Int32 CAS. HalfOpen allows exactly one probe goroutine. Carriers with Open breakers are excluded before goroutines are spawned.

⏳ Rate Limiting

Per-carrier token bucket via golang.org/x/time/rate. Wait() blocks and records rejections. TryAcquire() is non-blocking for hedge suppression.

📊 Prometheus Metrics

Isolated prometheus.NewRegistry() — never pollutes the global default. 8 metrics: CB state/transitions, hedges fired/won, rate limit rejections, quotes issued/failed, and latency histogram.

🗄 Quote Persistence

Optional PostgreSQL backend caches quotes by request_id. Background cleanup ticker purges expired quotes every 5 minutes. Graceful shutdown: ticker → HTTP drain → DB close.

🚚 Carriers

Three built-in mock carriers with distinct latency/reliability profiles, plus an optional real HTTP carrier:

Carrier Type Latency Failure Rate Coverage
Alpha Mock ~50ms 0% auto, homeowners
Beta Mock ~200ms ~10% auto, umbrella
Gamma Mock ~800ms 0% auto, homeowners, umbrella
Delta HTTP Variable Variable auto, homeowners

The adapter layer uses a generic Adapter[Req, Resp] interface with zero reflection. Register[Req, Resp] captures typed functions into type-erased closures stored in a thread-safe Registry.

🌐 API Endpoints

POST /quotes

Request a batch of quotes from all eligible carriers.

Request Body

{
  "request_id":      "req-001",
  "coverage_lines":  ["auto"],
  "timeout_ms":      5000        // optional, default 5000
}

Response (200 OK)

{
  "request_id":  "req-001",
  "duration_ms": 87,
  "quotes": [
    {
      "carrier_id":    "alpha",
      "carrier_ref":   "",
      "premium_cents": 62340,
      "currency":      "USD",
      "is_hedged":     false,
      "latency_ms":    52
    },
    {
      "carrier_id":    "beta",
      "premium_cents": 71200,
      "currency":      "USD",
      "is_hedged":     false,
      "latency_ms":    85
    }
  ]
}

Error Responses

400 — Invalid JSON, missing fields, unknown coverage line

422 — No eligible carriers for the requested coverage

504 — All carriers timed out

GET /metrics

Prometheus-format metrics. All metrics are prefixed with carrier_gateway_.

GET /healthz

Liveness probe. Returns 200 OK with body "ok".

🚀 Quick Start

# Build
go build ./cmd/carrier-gateway

# Run (defaults to :8080)
go run ./cmd/carrier-gateway

# Run on custom port
go run ./cmd/carrier-gateway -addr :9090

# With PostgreSQL persistence
DATABASE_URL=postgres://user:pass@localhost/carrier_gw go run ./cmd/carrier-gateway

# With Delta HTTP carrier
DELTA_BASE_URL=https://api.delta-insurance.example.com go run ./cmd/carrier-gateway

# Test (race detector required)
go test -race -count=1 ./...

🧰 Tech Stack

Language & Runtime

Go 1.25 — stdlib net/http with Go 1.22+ ServeMux method routing.

Concurrency

golang.org/x/sync/errgroup for fan-out. atomic.Int32 CAS for lock-free circuit breakers. atomic.Pointer CAS for EMA state.

Observability

prometheus/client_golang with isolated registry. Structured logging via log/slog (JSON output).

Persistence

PostgreSQL via database/sql + lib/pq driver. Optional — runs fully in-memory without a database.

Rate Limiting

golang.org/x/time/rate token bucket. Per-carrier isolation ensures one carrier's limits don't affect others.

Testing

Race detector (-race) on all tests. go.uber.org/goleak for goroutine leak detection. E2E tests via httptest.Server.
